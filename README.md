# UrbanStandz

**UrbanStandz** is a tech-driven property management and real estate solutions company based in Bengaluru. We provide comprehensive lifecycle management for residential, commercial, and industrial properties.

## Features

- **Modern & Responsive Design:** A beautiful, dynamic, and premium user interface built with React.
- **Service Offerings:** Detailed sections outlining our core competencies including Tenant Management, Financial Oversight, and Legal Support.
- **Client Testimonials:** A sleek, mobile-friendly carousel showcasing what our trusted clients say about us.
- **Contact Integration:** Seamless contact forms and clear support channels to get in touch with our team.

## Tech Stack

- **Framework:** React + Vite
- **Styling:** Custom CSS (with modern variables, glassmorphism, and responsive utilities)
- **Icons:** Lucide React

## Getting Started

To run this project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/urbanstandz.git
   cd urbanstandz
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. Open `http://localhost:5173` in your browser.
